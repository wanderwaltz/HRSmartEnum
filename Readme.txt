Данный репозиторий используется для создания, хранения и поддержки различных компонент, которые можно применять в нескольких проектах. Предполагается для этого применять git submodule.

git submodule добавляет ВЕСЬ репозиторий как сабмодуль, но позволяет выбрать ветку. Поэтому для каждой отдельной библиотеки/компонента необходимо создавать и поддерживать отдельную ветку в репозитории, чтобы при добавлении в проект не скачивать ничего лишнего, а только необходимую библиотеку. 

Соответственно, в мастер-ветке репозитория ничего храниться не должно.
